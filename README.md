## Install
`go get github.com/gorilla/websocket`  

## Run
`cd src/`  
`go run main.go`  
  
Launch [http://localhost:8000](localhost)  

## Run with hot-reload
https://github.com/canthefason/go-watcher  

Once instaled, run:  
`cd src/`  
`watcher`  
Enjoy!  

## Randword
`cd src/`  
`go run randWord/randWord.go`  

## Ping-Pong
`cd src/`  
`go run ping/main.go`  
`http://localhost:8000/ping.html`  
Enjoy!
